# 实践场景学习平台 — 产品需求文档（PRD）

## 1. 产品概述

### 1.1 产品定位

「实践场景学习平台」是面向职业院校的场景化教研管理与学习测评系统。平台支持教研管理员/教师构建面向真实岗位场景的实践任务链，配置多元化的学习资源与测评形式，并基于岗位能力点体系对学生的场景完成度进行胜任度分析。

### 1.2 目标用户

| 角色 | 核心职责 | 使用场景 |
| --- | --- | --- |
| 教研管理员 | 生产场景资源、管理审批流程与批次 | 场景编辑器、审批流程管理、批次管理 |
| 教师 | 消费场景资源（组织测评）、共建场景 | 场景评分管理、学生能力报告查看 |
| 学生 | 消费场景资源（学习+测评） | 任务学习、任务测评、能力报告查看 |

### 1.3 核心工作流

1.  **场景生产**：教研管理员/教师在「场景大厅」创建场景 → 编辑基础信息 → 配置任务链（每个任务配置8个步骤：基础信息/任务说明/知识点/能力点/资源/测评形式/评价规则/权重）→ 提交审批 → 审批通过后发布。
    
2.  **场景消费（教师端）**：查看已发布场景 → 教务系统排班排课（本期不做排课，默认固定账号查看） → 等待学生提交测评 → 按不同类型评价规则评分。
    
3.  **场景消费（学生端）**：学生进入场景任务 → 查阅学习资源与AI助手 → 完成测评（试卷/题库/现场问答/现场评审）→ 查看成绩与能力分析报告（本期不做能力分析报告）

## 3. 全局交互规范

### 3.1 页面布局框架

*   **顶部固定导航栏**：使用新的统一框架（顶部导航，包含门户首页、我的服务台、应用服务中心）
    
*   **左侧固定侧边栏**：使用新的统一框架，导航顶部包含返回按钮+平台标题，下方为导航菜单
    
*   **主内容区**：数据表格与表单填写区域
    
*   **场景编辑器特殊布局**：进入场景编辑（步骤 1基础信息/ 步骤 2任务编辑）时，页面采用全屏覆盖模式，隐藏侧边栏，顶部为 sticky 操作栏（上一步、下一步、保存、草稿、预览）。
    

### 3.4 场景对象状态流转

草稿(draft) → 提交审批 → 审批中(pending) → 审批通过→ 待发布(approved) → 发布→ 已发布(published)

                                                                     ↘ 审批驳回 → 已驳回(reject)→ 编辑→ 草稿(draft)

                                                                     ↘ 撤回审批 → 草稿(draft)

已发布(published) → 取消发布 → 待发布(approved) 

                              ↘ 编辑 → 草稿(draft)

待发布(approved) → 编辑 → 草稿(draft)

任意状态（draft/pending/approved/published）→ 克隆 → 草稿(draft)

除已发布外任意状态（draft/pending/approved）→ 删除

*   **草稿**：可编辑、可删除、可克隆、可提交审批。
    
*   **审批中**：可撤回审批、可删除、可克隆
    
*   **待发布：**可编辑、可发布、可删除、可克隆
    
*   **已驳回：**可编辑、可删除、可克隆
    
*   **已发布**：可取消发布、可克隆

## 4. 场景生产模块详细需求

### 4.1 场景管理（场景大厅）

**功能概述**：场景大厅是平台的资源管理中心，支持教研管理员和教师查看、筛选、管理自己创建/共建/公共的场景资源。支持列表视图和批次分组视图两种展示模式。

#### 4.1.1 页面布局

页面整体采用垂直堆叠布局，从上到下依次为：

1.  **顶部标题卡片**
    
2.  **视图切换区**（我的/共建/公共）
    
3.  **数据列表卡片**（含筛选栏+批量操作栏+场景列表）
    

#### 4.1.2 顶部标题卡片

*   **左侧**：
    
    *   标题：`场景管理`
        
    *   副标题：`维护场景信息、任务信息等场景资源管理功能`
        
*   **右侧操作按钮组**（从左到右）：
    
    1.  **配置审批流程**：点击打开「审批流程管理」弹窗。
        
    2.  **配置批次分组**：点击打开「批次分组管理」弹窗。
        
    3.  **导入资源包**：点击打开「导入资源包」弹窗。（本期暂不做）
        
    4.  **导入场景**：点击打开「导入场景」弹窗（支持 Excel/JSON，上传区域为虚线边框拖拽区）。
        
    5.  **新建场景**：点击跳转全屏编辑页面。
        
*   **统计面板**（仅在「我的场景」「共建场景」Tab 下显示）：
    
    *   5个 mini 卡片横向排列，每卡包含：指标名+ 数值
        
    *   指标：场景总数、未提交（draft）、审批中（pending）、已驳回（reject）、已发布（published）。
        

#### 4.1.3 视图切换区

*   **左侧 Tab**：`我的场景` / `共建场景` / `公共场景`
    
    *   切换 Tab 时，清空已选项、岗位筛选、批次筛选。
        
    *   「我的场景」：当前用户创建的场景。
        
    *   「共建场景」：当前用户在共建人列表中的场景。
        
    *   「公共场景」：全部场景，不展示统计面板。
        
*   **右侧视图模式切换**：`资源列表` / `批次分组`（参考岗位功能即可）
    

#### 4.1.4 筛选栏

位于数据列表卡片内顶部：

*   **搜索框**：placeholder `搜索场景名称 / 任务名称`，支持按场景名称和任务名称模糊搜索（搜索结果为场景列表）。
    
*   **岗位筛选**：选项分组展示，先按专业分组，再展示岗位。默认选项 `全部岗位`。
    
*   **批次筛选**：展示各批次名称。默认选项 `全部岗位`。
    
*   **状态筛选**：全部状态 / 草稿 / 审批中 / 已通过 / 已驳回 / 已发布。默认选项 `全部状态`。
    
*   **重置按钮**：清空所有筛选条件。
    

#### 4.1.5 批量操作栏

位于筛选栏下方，带分隔：

*   左侧文案：`已选择 N 项：`（有选中时高亮 ）/ `请选择场景：`（无选中时置灰）。
    
*   操作按钮，按业务逻辑启用/禁用：
    
    1.  **提交审批**：仅当选中项包含 `draft` 时可用。
        
    2.  **撤回审批**：仅当选中项包含 `pending` 时可用。
        
    3.  **发布**：仅当选中项包含 `approved` 时可用。
        
    4.  **取消发布**：仅当选中项包含 `published` 时可用。
        
    5.  **删除**：选中项不包含 `published` 时可用。
        
    6.  **克隆**：有选中即可用。
        
    7.  **调整批次分组**：有选中即可用
        
    8.  **导出**：有选中即可用。
        

> 批量操作执行后，清空选中态。

#### 4.1.6 场景列表（资源列表视图）

表格样式结构：

| 列名 | 最小展示宽度 | 备注 |
| --- | --- | --- |
| 复选框 | / |  |
| 场景名称 | 最短 15 字 | 点击进入场景编辑页面—步骤 1 |
| 场景编码 | 固定 10 字<br>资源类码规则：2位字母+数字随机组合 + 8 位随机递增数字 |  |
| 版本 | 最短 4 字 |  |
| 所属岗位 | 最短 8 字 |  |
| 所属批次分组 | 最短 12 字 |  |
| 创建人 | 最短 4 字 |  |
| 状态 | 最短 4 字 | 草稿/审批中/已驳回/待发布/已发布 |
| 发布时间 | 年-月-日 |  |
| 场景任务数量 | 最短 2 字 | 点击进入场景编辑页面—步骤 2 |
| 操作 | 鼠标移入后 hover 展示 |  |

*   **行交互**：鼠标 hover 时行背景变浅灰，选中行背景变蓝
    
*   **悬浮操作栏**：hover 行时出现，带白色半透明背景+阴影。操作按钮从左到右：
    
    1.  **查看详情**：跳转预览页面。
        
    2.  **编辑**（：跳转场景编辑页面—步骤 1
        
    3.  **编排任务**：跳转场景编辑页面—步骤 2。
        
    4.  **提交审批**：仅 `draft` 状态显示
        
    5.  **撤回审批**：仅 `pending` 状态显示
        
    6.  **发布**：仅 `approved` 状态显示
        
    7.  **取消发布**：仅 `published` 状态显示
        
    8.  **删除**：除了 `published` 状态外都显示
        
    9.  **查看驳回原因**：仅`reject`状态显示，打开驳回原因弹窗。
        
    10.  **克隆**：全部显示，触发克隆确认弹窗。
        
    11.  **调整批次分组**：全部显示，触发调整分组弹窗。
        

#### 4.1.7 批次分组视图

当视图模式为批次分组时：

*   按分组展示，每个批次分组为一个折叠面板。
    
*   面板头部：批次名称 + 编码 + 场景数量。
    
*   面板内容：该批次下的场景列表（复用场景列表样式）。
    
*   无批次的草稿场景单独放在底部「未分类」区域
    

#### 4.1.8 弹窗交互详情

**A. 新建场景弹窗**

*   触发：点击顶部「新建场景」按钮。
    
*   操作：进入完整场景编辑器（新建/编辑复用同一个编辑器）
    

**B. 批次分组管理弹窗**

*   宽度：内部嵌套「新增批次」弹窗。
    
*   内容：
    
    *   右上角「新增批次」按钮，打开内层弹窗。
        
    *   批次列表表格：分组名称、批次编号、审批流程、状态。
        
*   内层「新增批次」弹窗：
    
    *   字段：分组名称（输入）、批次编号（自动生成，disabled）、关联审批流）。
        
    *   操作：「取消」/「创建批次」
        

**C. 审批流程选择弹窗（无批次场景提交审批时）**

*   内容：审批流程列表选择（展示流程名称+步骤数），底部展示提交场景名称。
    
*   操作：「取消」/「提交审批」
    

**D. 导入场景弹窗**

*   内容：虚线边框拖拽上传区，提示「支持 .xlsx, .json 格式，单个文件不超过 10MB」。
    
*   操作：「取消」/「开始导入」
    

**E. 批量导出弹窗**

*   内容：导出格式选项卡片（Excel / JSON），点击选中。
    
*   操作：「取消」/「确认导出」
    

**F. 克隆重命名弹窗**

*   触发：点击单行/批量「克隆」后。
    
*   内容：输入框，默认值为 `{原名称} (克隆)`。
    
*   操作：「取消」/「确认克隆」。
    

**G. 驳回原因查看弹窗**

*   内容：展示驳回原因文本（从审批记录读取）。
    
*   操作：「关闭」。
    

**H. 调整批次分组弹窗**

*   触发：批量操作「调整批次分组」。
    
*   内容：批次选择列表。
    
*   操作：「取消」/「确定」。

### 4.2 场景编辑器 — 基础信息

**路由**：（新建）或（编辑）场景，同一个编辑器新页面

**功能概述**：场景编辑器第一步，用于配置场景的基础信息。页面采用全屏模式。

#### 4.2.1 顶部操作栏（Sticky）

*   **左侧**：
    
    *   「取消」按钮：点击返回场景大厅。
        
    *   步骤标识：显示「步骤 1」+ 「基础信息编辑」。
        
*   **右侧**：
    
    *   「保存草稿」。将当前场景数据保存（步骤 1+步骤 2 数据）
        
    *   「预览」：打开预览弹窗（使用学生端场景展示页面）。
        
    *   「下一步」：必填校验通过后可用，跳转步骤 2 页面。
        

> 必填校验规则：`场景名称` 非空。

#### 4.2.2 页面主体

**左栏主表单区**：

| 字段 | 组件 | 规则 |
| --- | --- | --- |
| 目标岗位 | 自定义复合选择器 | 点击展开浮层：含 Tabs（我的/共建/公共库）、搜索框、按专业分组展示岗位。选中后自动填充「适用专业」。非必填。 |
| 所属批次 | 选择器 | 非必填。 |
| 场景名称 | 输入框 | 必填，placeholder `请输入场景名称`。 |
| 场景编码 | 输入框 | 系统自动生成（`SC-XXXXXXXXXX`），disabled，不可修改。 |
| 面向行业 | 多选标签选择器 | 非必填，支持搜索，选中项以 Tag 形式展示，可逐个删除。 |
| 适用专业 | 多选标签选择器 | 非必填，同上面向行业组件。 |
| 难度等级 | 星级选择器 | 1-5 星，点击切换。对应文案：1入门/2基础/3中级/4高级/5专家。 |
| 场景介绍 | 富文本编辑器 | 含工具栏的富文本编辑器。placeholder 提示编写背景、意义和学习目标。 |

**右栏辅助信息区**：

| 区块 | 规则 |
| --- | --- |
| 场景封面 | 虚线边框上传区（`aspect-video`），placeholder：点击上传封面图片， 需要支持裁切功能 |
| 创建人 | 只读展示 `creatorName`。 |
| 共建人/共建部门 | 点击打开「选择共建人」穿梭框弹窗。已选人以 Tag 展示，可删除。 |
| 当前版本号 | 只读展示，根据规则自动升级。 |

#### 4.2.3 共建人选择弹窗

*   布局：左右分栏穿梭框。
    
*   **左侧（组织架构）**：
    
    *   搜索框（搜索用户或部门）。
        
    *   按部门折叠树：部门名 + 展开/折叠按钮 + 人数。
        
    *   展开后展示用户列表：复选框 + 用户名 + 角色标签（管理员/审批人/建设者）。
        
    *   点击行切换选中态。
        
*   **右侧（已选共建人）**：
    
    *   头部展示已选数量。
        
    *   已选用户列表：用户名 + 部门 + 删除按钮。
        
    *   空态提示：从左侧面板选择共建人。
        
*   操作：「完成」（关闭弹窗，无取消按钮，因为穿梭框是即时生效的）。

### 4.3 场景编辑器 — 任务链配置

**功能概述**：场景编辑器第二步，以「任务链」形式顺序罗列场景下的所有任务。每个任务通过 8 个配置卡片来定义学习+测评过程。任务链仅支持简单的顺序罗列，暂不支持分支功能。

#### 4.3.1 顶部操作栏（Sticky）

*   **左侧**：
    
    *   「 返回上一步」按钮：返回步骤 1 基础信息编辑页。
        
    *   分隔线 + 步骤标识：「步骤 2」+「任务链配置」。
        
*   **右侧**：
    
    *   「保存草稿」。将当前场景数据保存（步骤 1+步骤 2 数据）
        
    *   「预览」：打开预览弹窗（使用学生端场景展示页面）。
        
    *   「完成配置」：点击返回场景大厅，同时保存当前场景数据（步骤 1+步骤 2 数据）
        

#### 4.3.2 场景信息摘要卡片

展示当前场景的关键信息：

*   场景名称+ 共建状态（当步骤 1 有共建人的时候，自动设为共建状态）。
    
*   岗位名 `|` 行业名 `|` 专业名 + 共建人列表（Tag 形式）。
    
*   右侧：难度星级 + 版本号。
    
*   底部：场景背景介绍文本。
    

#### 4.3.3 任务列表操作栏

*   **左侧**：
    
    *   「任务列表」标题 + 任务数量。
        
    *   权重校验提示：展示当前所有任务权重加总值。当所有任务权重加总=100% 时绿色背景；否则黄色背景，提示需调整。
        
*   **右侧按钮**：
    
    1.  **添加任务**：打开「添加任务」弹窗。
        
    2.  **克隆/引用**：打开「克隆/引用任务」弹窗。
        
    3.  **配置任务权重**：打开「权重配置」弹窗。
        

#### 4.3.4 任务列表展示

采用横向表格布局，支持水平滚动：

*   **列头行**：8 个配置卡片的标题横向排列，每个标题宽度固定，居中对齐
    
*   **任务行**：每个任务占一行，从左到右：
    
    *   **序号区**：拖拽图标+ 序号数字，任务顺序可上下拖动。
        
    *   **8 个配置卡片**：
        
        *   卡片为 `button` 元素，可点击打开编辑弹窗。
            
        *   卡片头部：图标 + 卡片标题 + 「引用标记」（当任务为引用状态时展示）
            
        *   卡片内容：配置摘要文本，已配置根据下面字段展示，未配置缺省文本“未配置”。
            
        *   引用任务的所有卡片（除 weight 外）置灰，不可点击。
            
    *   **删除按钮**，在每行任务的最右侧，点击之后二次确认弹窗后删除）。
        

**8 个配置卡片定义**：

| 标题 | 卡片摘要规则 |
| --- | --- |
| 配置任务基础信息 | 展示任务名称、编码、类型、难度、预估学时 |
| 配置任务说明 | 展示描述前50字，未填写展示「未填写」 |
| 考查知识点 | 展示已选知识点名称（最多5个），未选展示「未配置」 |
| 考查能力点 | 展示已选能力点名称（最多5个），未选展示「未配置」 |
| 配置任务资源 | 展示已选资源名称（最多5个），未选展示「未配置」 |
| 配置任务测评形式 | 展示已选测评形式中文名（顿号分隔），未选展示「未配置」 |
| 配置任务评价规则 | 展示各测评形式权重占比，以及「待配置」/「已配置」状态 |
| 配置任务权重 | 只读展示，任务权重百分比 |

#### 4.3.5 添加任务弹窗

*   字段：
    
    *   任务名称（必填）。
        
    *   任务类型（训练任务 / 考核任务。
        
    *   预估学时（数字，附带说明文字「学生完成任务的预估时长」。
        
    *   难度（星级选择器，1-5星）。
        
    *   背景介绍（纯文本输入框，预留 3 行高度）。
        
*   操作：「取消」/「添加」（名称必填）。
    

#### 4.3.6 克隆/引用任务弹窗

*   顶部：模式切换按钮「克隆（可编辑）」/「引用（只读）」。
    
*   搜索框：按任务名称、编码、关联场景搜索。
    
*   Tab：我的 / 共建 / 公共库。
    
*   列表：表格展示（任务名称、编码、关联场景、关联岗位），点击行切换复选框选中态。
    
*   操作：「取消」/「确定」。
    

#### 4.3.7 权重配置弹窗

*   弹窗内展示所有任务的权重输入框 + 锁定开关。
    
*   「一键平均分配」按钮：按未锁定任务平均分配权重，确保总和为100%。
    
*   实时展示总权重，不等于100%时黄色提示。

### 4.4 场景编辑器 — 任务配置卡片（8 个步骤）

点击任务行中的配置卡片后，打开「编辑卡片」弹窗。弹窗宽度根据内容自适应。

弹窗通用结构：

*   **头部**：卡片标题+简短说明。
    
*   **主体**：根据卡片类型展示不同表单内容，支持垂直滚动。
    
*   **底部**：「取消」/「保存」。
    

以下分别描述 8 个卡片的详细交互。

#### 4.4.1 配置任务基础信息（info）

*   任务名称，纯文本输入框，1行。
    
*   任务类型，训练 / 考核。
    
*   预估学时，附带说明文案。
    
*   难度，星级选择器，1-5星。
    
*   背景，纯文本输入框，3行。
    

#### 4.4.2 配置任务说明（description）

*   **Tab 切换**：`富文本编辑` / `上传任务说明书`。
    
*   **富文本编辑**：
    
    *   工具栏：含标题（H1/H2）、正文、加粗(B)、斜体(I)、下划线(U)、删除线、左/中/右对齐、无序/有序列表、引用、代码、链接、图片、视频、表格、分割线、字体颜色、背景色。
        
    *   编辑区：带默认模板文案（任务描述/任务目标/任务结果/测评要求），输入后不消失。
        
*   **PDF 上传**：
    
    *   虚线边框上传区，支持点击/拖拽。
        
    *   已上传后展示 PDF 图标 + 文件名 + 「移除文件」按钮（只允许上传 1 个）。
        

#### 4.4.3 考查知识点（knowledge）

采用左右分栏布局（左 3/5，右 2/5）。

*   **顶部搜索栏**：搜索知识点名称/描述/编码，右侧「新增知识点」按钮。
    
*   **左侧（知识点库列表）**：
    
    *   表格表头：知识点名称、编码、描述、操作。
        
    *   表格行：名称 + 编码 + 描述+ 操作按钮组。
        
    *   操作按钮：
        
        *   「详情」：打开知识点详情弹窗。
            
        *   未选中时：「引用」（加入右侧已选）、「克隆」（基于该知识点创建副本并加入已选）。
            
        *   已选中时：「取消」（从右侧移除）。
            
    *   搜索无结果时，展示提示「未找到 XXX 相关的知识点」+「新增此知识点」按钮。
        
*   **右侧（已选知识点）**：
    
    *   标题：已选择知识点 ({count})。
        
    *   网格卡片：每张卡片展示名称、描述、关联颗粒课 Tag（最多3个，超出显示 `+N`）。
        
    *   引用知识点：右下角灰色角标「引用」；自定义知识点：无角标，点击打开编辑弹窗。
        
    *   每张卡片右上角有删除按钮（`X`）。
        

**知识点详情弹窗**：

*   展示：名称、知识点描述、编码、关联颗粒课列表。
    
*   引用知识点标注「引用（不可编辑）」；自定义知识点标注「自定义（可编辑）」，并支持「引用颗粒课」「新增颗粒课」按钮。
    

**新增/克隆/编辑知识点弹窗**：

*   字段：知识点名称、描述、编码（新增/克隆时自动生成不可改，编辑时可改）、关联颗粒课（Tag 展示 + 「选择颗粒课」按钮 + 「新建颗粒课」外链按钮）。
    
*   操作：「取消」/「新增并选中」或「克隆并选中」或「保存修改」。
    

**颗粒课选择弹窗**：

*   左右分栏：左侧全部颗粒课列表（搜索+复选框），右侧已选列表。
    
*   操作：「确定」。
    

#### 4.4.4 考查能力点（ability）

*   若场景未关联岗位，展示警告提示「请先关联岗位后，再选择考察能力点」。
    
*   **顶部搜索栏**：搜索能力点名称/编码/描述，右侧展示统计「共 N 个关联能力点，已选 M 个」。
    
*   **能力点分组展示**：参考岗位平台，按「能力领域」分组「能力点」的卡片。
    
    *   每个分组为可折叠卡片，头部：领域图标 + 领域名 + 能力点数量。
        
    *   展开后：能力点列表，每行含复选框、名称、编码、掌握程度 （了解/理解/掌握/熟练/精通，各有颜色）、描述、关联岗位 Tag。
        
    *   点击整行切换选中态。已选中行左侧带高亮。
        
*   搜索无结果时展示空态图标+提示。
    
*   **能力点详情弹窗**：展示名称、描述、所属领域、关联岗位、掌握程度要求、熟练程度描述。
    

> 能力点数据来源于岗位功能，不从本模块维护。

#### 4.4.5 配置任务资源（resources）

采用左右分栏布局（左 4/5，右 1/5）。

*   **顶部工具栏**：
    
    *   资源类型筛选标签（横向排列按钮）：全部、文档、表格、图片、链接、音频、视频、压缩包、场地、设施设备、软件、其他。选中态高亮。
        
    *   搜索框：资源名称、资源提供者，中间夹「重置」按钮。
        
    *   「上传资源」按钮。
        
*   **左侧（资源库网格）**：
    
    *   网格卡片，每张卡片：顶部缩略图区（文件类型图标+类型 Badge）+ 选中角标+ 信息区（名称、提供者、日期）+ 底部操作区（「预览」外链按钮、「选择/取消」按钮）。
        
    *   点击卡片非按钮区域也切换选中态。
        
*   **右侧（已选资源列表）**：
    
    *   标题 + 数量。
        
    *   列表项：类型图标 + 名称 + 提供者·日期 + 删除按钮。
        
    *   空态：提示「请从左侧选择资源」。
        

**上传资源类型选择弹窗**：

*   网格展示 11 种资源类型图标+名称，点击后进入上传弹窗并预填类型。
    

**上传资源弹窗**：

*   通用字段：资源名称（必填）、资源描述。
    
*   根据资源类型展示特定字段（参考原型）：
    
    *   **链接**：URL 地址。
        
    *   **场地**：场地地址、开放时间、容纳人数、联系人/电话。
        
    *   **设施设备**：所在位置、数量。
        
    *   **软件**：版本号、下载链接、授权信息。
        
    *   **文件类**（文档/表格/图片/音频/视频/压缩包/其他）：文件上传拖拽区（提示「支持多种格式，最大 100MB」）。
        
*   操作：「取消」/「上传并选中」（名称必填）。
    

#### 4.4.6 配置任务测评形式（evaluation）

*   按分类分组展示（基础考核、综合评估、互动评价、智能评测），每组一个浅色背景卡片区。
    
*   每个测评形式以卡片按钮展示：
    
    *   左侧：图标 + 名称+ 描述。
        
    *   右侧：
        
        *   「查看介绍」按钮（书本图标，点击新开标签页跳转）。
            
        *   已开通时：可选中的按钮状态
            
        *   未开通时：不可选中状态，卡片整体灰色，中央盖印「未开通」水印。
            
    *   点击卡片切换选中/取消选中（仅对已开通有效）。
        
*   本期仅 4 种测评形式为：**现场问答**、**现场评审**、**试卷**、**题库**。其余均显示未开通状态。
    
*   底部空态：未选任何测评形式时展示提示「请选择至少一种评价方式」。
    

#### 4.4.7 配置任务评价规则（evaluationRules）

> 本节为最复杂配置模块，包含评价点管理、评分主体配置、各测评形式的细化规则。

**页面结构**：纵向堆叠多行卡片+模块。

**测评形式配置概览**

*   对当前任务已选的每种测评形式，展示一个概览卡片：
    
    *   测评对象+评价主体+测评资源+评价标准配置
        
    *   点击卡片展开该测评形式下面每个步骤的详细配置面板。
        

**A. 测评对象配置（全局一致）** 

*   2 种测评对象，个人或者小组，目前无实际功能区分，仅作数据记录
    

**B. 评分主体配置（全局一致）**

*   6 种评分主体，每个主体一行：
    
    1.  **教师**：教师专业背景、评分人数（超过 1 人时，配置记分规则：平均/中位数/最高/最低）、权重占比、最低教龄。
        
    2.  **企业导师**：专业领域、最低工作年限、岗位工作经历、评分人数（超过 1 人时，配置记分规则：平均/中位数/最高/最低）、权重占比。
        
    3.  **自评**：是否需要反思、反思最低字数、权重占比。
        
    4.  **互评**：互评人数、互评规则、是否匿名、权重占比。
        
    5.  **AI 评价**：AI 模型、权重占比。
        
    6.  **服务对象**：服务方式、样本量、权重占比。
        
*   每行主体使用 开关 控制启用/禁用，启用后展示参数配置区。
    

**C. 各测评形式测评资源配置**

*   **现场问答（random\_draw）**：
    
    *   选题弹窗：题库列表表格（参考原型，题目数据来源于测评认证中心），支持搜索，勾选题目加入已选。
        
    *   抽题规则：题目数量、难度分布、是否自动抽题。
        
*   **现场评审（review）**：
    
    *   评审材料要求：评审场地/环境资源准备、提交材料要求、截止天数、是否提交材料、允许重复提交。
        
    *   评审步骤配置：默认「初评」「复评」「终评」。每步可配置：步骤名称、描述、评审主体类型、权重。支持新增/编辑/删除步骤。
        
*   **试卷（paper）**：
    
    *   试卷选择：从试卷库选择（展示试卷名称、题数、总分）。
        
    *   新建试卷：跳转测评认证中心配置试卷。
        
    *   详情查看：跳转测评认证中心查看试卷。
        
    *   参数配置：考试时长、及格分数、是否允许重考、重考次数、是否乱序、是否展示成绩、激活方式（手动/定时/永久开放）、定时时间。
        
*   **题库（question\_bank）**：
    
    *   选题弹窗：题库列表，支持搜索，勾选题目。并可以给每道选中题目配置分值。
        
    *   抽题规则：题目数量、难度分布、总分、是否自动生成、时间限制、题型权重、是否允许重考、乱序、展示成绩。
        

**D. 评价标准配置管理**

*   每个测评形式独立维护评价点列表（试卷和题库自动记分，因此不需要配置，现场问答和现场评审共用同一套方式）。
    
*   评价标准配置弹窗（`EvalPointCard`）：
    
    *   名称输入框（可编辑）。
        
    *   评价标准类型：评价量规、评分规则。
        
    *   评价量规配置表：
        
        *   评价维度：一句话的方式描述评价维度，和下面两个选择合并为同一个弹窗
            
        *   关联能力点：展示已关联能力点 Tag（可删除）+「+ 添加能力点」按钮（打开能力点搜索弹窗）。
            
        *   关联知识点：展示已关联知识点 Tag（可删除）+「+ 添加知识点」按钮（打开知识点搜索弹窗）。
            
        *   等级转换规则编辑器：等级名称输入（如 A/B/C/D）、颜色圆点、分数区间（min-max 数字输入）、等级备注（一句话辅助教师参考），可新增删除修改等级。
            
        *   权重配置：给每条评价维度配置权重
            
        *   删除。
            
        *   新增评价维度按钮：底部输入框输入名称后按回车或点击添加。
            
        *   一键均分：将所有评价维度的权重进行均分
            
    *   评价规则配置表：
        
        *   评价项/标准描述：一句话的方式描述评价标准
            
        *   加减分规则编辑器：一句话辅助教师参考加减分打分
            
        *   分值配置：给每条评价项配置分值
            
        *   删除。
            
        *   新增评价项按钮：底部输入框输入名称后按回车或点击添加。
            
        *   一键均分：将所有评价项的分值进行均分（100 分总分）
            
*   评价标准模板功能：
    
    *   可将上面的评价量规表或者评分规则配置表保存为模板
        
    *   支持模板一键覆盖、保存为模板、删除模板等功能
        

#### 4.4.8 配置任务权重

*   在任务列表的 weight 卡片中只读展示百分比。（）
    
*   点击「配置任务权重」按钮打开弹窗：
    
    *   每个任务一行：任务名称 + 权重数字输入框 + 锁定开关。
        
    *   锁定任务在「均匀分配」时权重保持不变。
        
    *   底部总权重实时计算，不等于 100% 时黄色高亮提示。

### 4.6 审批管理

**功能概述**：审批中心供有审批权限的用户查看待审批场景、执行通过/驳回操作，以及查看历史审批记录。

#### 4.6.1 页面布局

*   **Tabs**：`待审批` / `已审批`。

*   表身：每行展示对应字段，状态展示（待审批-yellow、已通过-green、已驳回-red）。
    
*   操作列按钮：
    
    *   「查看」：跳转场景预览页（参考学生的学习页面）

*   操作：「取消」/「确认通过」。
    

#### 4.6.4 驳回确认弹窗

*   内容：驳回原因 （必填）。
    
*   操作：「取消」/「确认驳回」。

### 4.10 审批流程模板管理

**功能概述**：教研管理员预设校内审批流模板，供批次关联使用。

#### 4.10.1 页面布局

*   页面标题：`审批流程管理`，副标题 `预设校内审批流模板，供批次关联使用`。
    
*   右上角「新增审批流程」按钮（主按钮，图标 `Plus`）。
    

#### 4.10.2 审批流程列表（参考使用现有岗位中的功能即可）

*   表格：流程名称、流程说明、审批步骤数、创建日期、操作。
    
*   操作列：
    
    *   「编辑」：打开编辑弹窗。
        
    *   「删除」： 点击后确认删除。
        

#### 4.10.3 新增/编辑审批流程弹窗（参考使用现有岗位中的功能即可）

*   字段：
    
    *   流程名称，必填。
        
    *   流程说明，3行。
        
    *   审批步骤：动态列表，每步包含：
        
        *   序号 Badge（圆形）。
            
        *   步骤名称
            
        *   审批角色
            
        *   删除按钮（步骤数大于1时显示）。
            
    *   「添加步骤」按钮
        
*   操作：「取消」/「保存」（名称必填，且至少保留一个有效步骤）。

## 5. 场景消费模块详细需求

### 5.1 场景评分模块（教师端）

**功能概述**：教师查看已发布场景下的学生提交，按任务和测评形式维度进行评分管理。

#### 5.1.1 页面布局（左右分栏）

*   **左侧（场景选择列表）**：宽度固定，可按岗位分组折叠展示场景。
    
    *   搜索框：按场景名称/编码搜索。
        
    *   岗位分组：岗位名+ 场景数量 。
        
    *   每个场景卡片：场景名称 + 编码 + 统计（任务数、待评分、已评分、学生数）。
        
    *   点击场景卡片选中，右侧展示该场景的任务分组。
        
*   **右侧（任务评分区）**：
    
    *   场景名称标题 + 基本信息 Badge（岗位、批次、编码）。
        
    *   任务列表：每个任务为一个折叠面板。
        
        *   头部：任务名 + 任务类型（训练/考核）+ 测评形式 + 待评分/已评分数量。
            
        *   展开后：按测评形式分 Tab 展示学生提交列表。
            
            *   学生提交列表表格：学生姓名、学号、班级、入学年份、提交状态、操作（进入评分）。
                
            *   操作按钮：「评分」跳转不同类型的评分详情页
                

#### 5.1.2 任务评分详情页

*   展示该任务下所有学生的提交，支持按测评形式筛选。
    
*   每种测评形式对应不同的评分界面：
    
    *   **试卷/题库**：展示学生客观题答案与正误对比，主观题需教师人工评分。
        
    *   **现场问答**：记录教师现场抽题和学生回答情况，按评价点打分。
        
    *   **现场评审**：查看学生提交的材料，按评审量规的多维度评价点打分。

### 5.2 场景学习测评模块（学生端）

#### 5.2.1 功能概述

场景学习页面是学生端的核心消费入口，展示已发布场景的完整信息，支持学生浏览场景详情、查看任务链、查阅学习资源与配套材料，并进入具体任务进行学习+测评。页面采用「场景详情视图」与「任务学习视图」两层结构，详情视图以 Tab 形式聚合场景多维信息，学习视图则以单页双面板形式呈现任务的学习内容与测评内容。

#### 5.2.2 页面结构

1.  **场景详情视图**（全局视图）：场景信息卡片 + 课时统计侧边栏 + Tab 标签内容区(任务概览+资源中心+能力模型+评价标准+知识图谱)。
    
2.  **任务学习视图**（点击任务后进入）：顶部任务信息栏 + 学习/测评双面板 + 全局资源侧边栏。
    

#### 5.2.3 场景详情视图

##### A. 场景信息卡片

采用左图右文的横向卡片布局：

*   **左侧封面区**：场景封面图（或默认图标占位），圆角矩形。
    
*   **右侧信息区**：
    
    *   标题行：场景名称+ 版本号 （如 V1.0）+ 操作按钮组（分享、收藏）。
        
    *   元信息行：创建者及单位 | 更新时间 | 浏览次数 | 收藏次数。
        
    *   场景描述：场景背景与学习目标简介，多行文本，默认展开。
        
    *   标签行（横向排列）：
        
        *   适用专业：以蓝色 Tag 展示。
            
        *   关联岗位：以紫色 Tag 展示。
            
        *   关联行业：以橙色 Tag 展示。
            
    *   共建人/单位行：以灰色 Tag 展示共建单位名称及图标。
        

##### B. 课时统计侧边栏

位于场景信息卡片右侧，固定宽度卡片：

*   **环形图**：中心展示总课时数，外环以两种颜色区分「考核课时」与「训练课时」占比。
    
*   **图例**：考核课时（理论色）+ 训练课时（实践色）及对应数值。
    

##### C. 标签页导航

位于场景信息下方，横向 Tab 排列：

| Tab 名称 | 角标 | 内容说明 |
| --- | --- | --- |
| 任务概览 | 任务数量 | 场景下所有任务的列表卡片 |
| 资源中心 | 资源数量 | 聚合所有任务的配套资源，按类型分组 |
| 能力模型 | 能力点数量 | 按能力领域分组展示关联能力点 |
| 评价标准 | 评价方式数量 | 任务权重分配表格 |
| 知识图谱 | 知识点梳理 | 场景-任务-知识点关系可视化图谱 |

*   切换 Tab 时，下方内容区平滑切换，保持 URL 不变。
    
*   当前激活 Tab 带下划线高亮，Badge 为灰色圆角背景。
    

##### D. 任务概览（默认 Tab）

以纵向卡片列表展示场景下的所有任务，每个任务卡片包含：

*   **任务内容区**：
    
    *   标题行：任务名称 + 任务编码 Badge + 任务类型标签（考核任务/训练任务，颜色区分）。
        
    *   任务描述：一行简介文本。
        
    *   测评方式标签行：根据任务配置的测评形式展示对应图标+名称标签（试卷/题库/现场问答/评审）。
        
    *   元信息行：预估学时 | 难度星级 | 关联知识点数量 | 关联能力点数量 | 资源数。
        
*   **操作按钮区**：
    
    *   点击任务卡片整体区域或开始学习按钮，进入「任务学习视图」。
        

##### E. 资源中心

聚合场景下所有任务的配套资源，按资源类型分组展示：

*   资源类型分组标题：文档资料、视频教程、参考链接、软件工具、图片资源、场地资源、其他资源。
    
*   每组内以网格/列表展示资源卡片：资源类型图标 + 资源名称 + 大小/链接类型 + 操作按钮（预览/打开外部链接）。
    
*   无资源时展示空态：图标 + 「暂无学习资源」文案。
    

##### F. 能力模型

收集场景下所有任务关联的能力点，按「能力领域」分组折叠展示：

*   分组头部：领域图标 + 领域名称。
    
*   展开后：能力点列表，每行含能力点名称、编码、类型标签（技能）。
    
*   点击能力点可打开知详情弹窗。
    

##### G. 评价标准

以折叠表格形式展示各任务在场景中的权重分配：

*   表头：序号、任务名称、任务编码、任务类型、场景权重。
    
*   表身：每行对应一个任务，权重以百分比展示。折叠后可以看到改任务下面的多种测评方式权重
    

##### H. 知识图谱

采用可视化画布展示场景-任务-知识点的层级关系网络：

*   **节点类型**：场景名称（紫色）、任务节点（绿色，可点击）、知识点节点（黄色，可点击）。
    
*   **交互操作**：
    
    *   拖拽节点调整位置。
        
    *   滚轮缩放画布。
        
    *   点击任务/知识点节点弹出详情卡片。
        
*   **工具栏**：放大、缩小、重置视角按钮。
    
*   **图例**：展示三种节点类型的颜色说明。
    
*   **提示文案**：画布底部显示操作提示（拖拽节点 · 滚轮缩放 · 点击查看详情）。
    

#### 5.2.4 任务学习视图

点击任务后，场景详情视图隐藏，切换为任务学习视图。采用单页双面板（上下排列）布局。

##### A. 顶部任务信息栏

*   **返回场景按钮**：左侧，带返回箭头图标，点击返回场景详情视图。
    
*   **任务名称**：大字号加粗展示。
    
*   **任务元信息**：预估学时 · 难度星级 · 任务类型（考核/训练），右对齐。
    

三个元素纵向排列：返回按钮独占第一行，任务名称与元信息在同一行紧随其后。

##### B. 学习面板（左侧主区域）

以纵向卡片流展示任务的学习内容：

1.  **任务说明卡片**：
    
    *   卡片头部：「任务说明」标题 + 全屏查看按钮。
        
    *   卡片内容：富文本渲染的任务详细描述/背景/目标/交付标准，带垂直滚动条（最大高度 420px）。
        
    *   全屏弹窗：点击全屏按钮后以遮罩弹窗形式展示完整任务说明，支持关闭。
        
2.  **知识点标签**：以可点击 Tag 形式展示本任务关联的知识点，点击后弹出知识点详情卡片。
    
3.  **能力点标签**：以可点击 Tag 形式展示本任务关联的能力点，点击后弹出能力点详情卡片。
    
4.  **交付要求**：
    
    *   标题：「交付要求」。
        
    *   列表：每个交付物为一个编号条目，含交付物名称 + 描述。
        
    *   现场评审类型交付物：带特殊图标标识。
        
5.  **配套资源**：
    
    *   按资源类型分组展示，同「资源中心」的卡片样式。
        
    *   支持预览视频/文档、打开外部链接。
        

##### C. 测评面板（与学习面板上下排列）

当任务配置了测评形式时，在学习面板下方以分割线隔开展示测评内容：

*   **测评标题**：「任务测评」+ 图标。
    
*   **测评头部信息**：共 N 道题目，总分 X 分 + 测评类型标签（试卷/题库）。
    
*   **已作答状态**：已提交后展示「已作答」绿色标识。
    

**各测评形式渲染规则：**

| 测评形式 | 交互说明 |
| --- | --- |
| 试卷测评 | 展示固定试卷题目（单选/多选/判断），每题含选项、分值。已提交后展示正误对比与正确答案。 |
| 题库测评 | 从题库抽取题目，交互形式同试卷测评。 |
| 现场问答 | 展示预约提示：需按预约时间参加，教师将现场抽题提问。 |
| 教师评审 | 展示提交材料要求与评审流程说明，提示在交付步骤中上传成果。 |
| 综合测评 | 当任务配置多种测评形式时，以网格卡片分别展示各测评形式的入口与说明。 |

**题目卡片交互（试卷/题库）：**

*   每题为独立卡片，含题号、题干、分值。
    
*   选项为可点击行，选中后显示勾选标记。
    
*   已提交后：正确选项绿色边框+背景，错误选项红色边框+背景，底部显示反馈文案（回答正确/回答错误 + 正确答案）。
    
*   未提交时底部居中展示「确认答案」主按钮。
    

##### D. 底部提交栏

位于测评面板下方，根据提交状态展示不同内容：

**未提交状态：**

*   提交内容检查区：列出各测评形式的完成状态摘要（客观题作答状态、成果材料状态、现场问答状态）。
    
*   学习进度：已完成步骤数 / 总步骤数。
    
*   警告提示：黄色背景卡片，提示提交后进入「待评分」状态，无法再次修改。
    
*   「确认提交给教师评分」主按钮。
    

**已提交状态：**

*   成功图标 + 「已提交给教师评分」标题。
    
*   说明文案：提示已转交教师评分，可在学习记录中查看状态。
    
*   提交内容摘要卡片：展示各项测评的提交状态与提交时间。
    

##### E. 全局资源侧边栏

右侧固定滑出面板，默认收起，通过右下角标签栏触发：

*   **面板头部**：标题（配套资源 / 知识点 / 任务）+ 关闭按钮。
    
*   **面板内容**：根据当前激活的 Tab 动态渲染对应的资源/知识点/任务列表。
    
*   **标签栏**（固定在页面右下角）：三个图标按钮——资源、知识、任务，点击切换侧边栏内容并展开面板。
    

#### 5.2.5 弹窗交互详情

##### A. 任务说明全屏弹窗

*   **触发**：点击任务说明卡片的「全屏查看」按钮。
    
*   **内容**：以遮罩层 + 居中弹窗形式展示完整任务说明，弹窗宽度最大 900px，高度 90vh，支持垂直滚动。
    
*   **操作**：点击遮罩层或关闭按钮退出全屏。
    

##### B. 知识点详情卡片

*   **触发**：点击任务学习面板中的知识点标签，或知识图谱中的知识点节点。
    
*   **内容**：知识点名称、描述、编码、关联颗粒课列表。
    
*   **操作**：点击卡片外部关闭。
    

##### C. 能力点详情卡片

*   **触发**：点击任务学习面板中的能力点标签，或知识图谱中的能力点节点。
    
*   **内容**：能力点名称、描述、所属领域、关联岗位、掌握程度要求。
    
*   **操作**：点击卡片外部关闭。
    

##### D. 资源预览弹窗

*   **触发**：点击资源卡片上的「预览」按钮。
    
*   **内容**：视频播放器或文档预览器（本期以占位提示实现）。
    
*   **操作**：「关闭」按钮。
    

##### E. 学测模式选择弹窗

*   **触发**：点击任务列表的「开始学习」按钮（当任务配置了 simulation 时）。
    
*   **内容**：学测一体 / 仅学习 / 仅测评 三种模式选择卡片。
    
*   **操作**：选择模式后进入对应视图，或取消关闭弹窗。
