### 5.2 场景学习页面

#### 5.2.1 功能概述

场景学习页面是学生端的核心消费入口，展示单个已发布场景的完整信息，支持学生浏览场景详情、查看任务链、查阅学习资源与配套材料，并进入具体任务进行学习+测评。页面采用「场景详情视图」与「任务学习视图」两层结构，详情视图以 Tab 形式聚合场景多维信息，学习视图则以单页双面板形式呈现任务的学习内容与测评内容。

#### 5.2.2 页面布局

页面整体采用上下布局，从上到下依次为：

1. **顶部固定页眉**：平台 Logo + 返回首页按钮 + 日期时间 + 快捷导航（新建场景 / 我的场景）。
2. **面包屑导航**：首页 / 场景库 / 当前场景名称。
3. **场景详情视图**（默认态）：场景信息卡片 + 课时统计侧边栏 + Tab 标签内容区。
4. **任务学习视图**（点击任务后切换）：顶部任务信息栏 + 学习/测评双面板 + 底部提交栏 + 全局资源侧边栏。

#### 5.2.3 场景详情视图

##### A. 场景信息卡片

采用左图右文的横向卡片布局：

- **左侧封面区**：场景封面图（或默认图标占位），圆角矩形。
- **右侧信息区**：
  - 标题行：场景名称（H1）+ 版本号 Badge（如 V1.0）+ 操作按钮组（分享、收藏）。
  - 元信息行：创建者及单位 | 更新时间 | 浏览次数 | 收藏次数。
  - 场景描述：场景背景与学习目标简介，多行文本，默认展开。
  - 标签行（纵向排列）：
    - 适用专业：以蓝色 Tag 展示。
    - 关联岗位：以紫色 Tag 展示。
    - 关联行业：以橙色 Tag 展示。
  - 共建人/单位行：以灰色 Tag 展示共建单位名称及图标。

##### B. 课时统计侧边栏

位于场景信息卡片右侧，固定宽度卡片：

- **环形图**：中心展示总课时数，外环以两种颜色区分「考核课时」与「训练课时」占比。
- **图例**：考核课时（理论色）+ 训练课时（实践色）及对应数值。

##### C. 标签页导航

位于场景信息下方，横向 Tab 排列：

| Tab 名称 | 图标 | Badge | 内容说明 |
|---------|------|-------|---------|
| 任务概览 | fa-tasks | 任务数量 | 场景下所有任务的列表卡片 |
| 资源中心 | fa-book | 资源数量 | 聚合所有任务的配套资源，按类型分组 |
| 能力模型 | fa-star | 能力点数量 | 按能力领域分组展示关联能力点 |
| 评价标准 | fa-clipboard-check | - | 任务权重分配表格 |
| 知识图谱 | fa-sitemap | - | 场景-任务-知识点关系可视化图谱 |

- 切换 Tab 时，下方内容区平滑切换，保持 URL 不变。
- 当前激活 Tab 带下划线高亮，Badge 为灰色圆角背景。

##### D. 任务概览（默认 Tab）

以纵向卡片列表展示场景下的所有任务，每个任务卡片包含：

- **任务内容区**：
  - 标题行：任务名称 + 任务编码 Badge + 任务类型标签（考核任务/训练任务，颜色区分）。
  - 任务描述：一行简介文本。
  - 测评方式标签行：根据任务配置的测评形式展示对应图标+名称标签（试卷/题库/现场问答/评审）。
  - 元信息行：预估学时 | 难度星级 | 关联知识点数量 | 关联能力点数量 | 资源数。
- **操作按钮区**：
  - 根据任务进度与提交状态动态展示按钮文案与样式：
    - 未开始：「开始学习」主按钮。
    - 进行中：「继续学习 (X%)」次要按钮。
    - 已完成：「已完成」置灰按钮。
    - 已提交：「待评分」次要按钮。
    - 已评分：「已评分」置灰按钮。
  - 点击任务卡片整体区域或按钮，进入「任务学习视图」。

##### E. 资源中心

聚合场景下所有任务的配套资源，按资源类型分组展示：

- 资源类型分组标题：文档资料、视频教程、参考链接、软件工具、图片资源、场地资源、其他资源。
- 每组内以网格/列表展示资源卡片：资源类型图标 + 资源名称 + 大小/链接类型 + 操作按钮（预览/打开外部链接）。
- 无资源时展示空态：图标 + 「暂无学习资源」文案。

##### F. 能力模型

收集场景下所有任务关联的能力点，按「能力领域」分组折叠展示：

- 分组头部：领域图标 + 领域名称。
- 展开后：能力点列表，每行含能力点名称、编码、类型标签（技能）。
- 点击能力点可打开知识图谱详情弹窗。

##### G. 评价标准

以表格形式展示各任务在场景中的权重分配：

- 表头：序号、任务名称、任务编码、任务类型、场景权重。
- 表身：每行对应一个任务，权重以百分比展示。

##### H. 知识图谱

采用 SVG 可视化画布展示场景-任务-知识点的层级关系网络：

- **节点类型**：场景名称（紫色）、任务节点（绿色，可点击）、知识点节点（黄色，可点击）。
- **交互操作**：
  - 拖拽节点调整位置。
  - 滚轮缩放画布。
  - 点击任务/知识点节点弹出详情卡片。
- **工具栏**：放大、缩小、重置视角按钮。
- **图例**：展示三种节点类型的颜色说明。
- **提示文案**：画布底部显示操作提示（拖拽节点 · 滚轮缩放 · 点击查看详情）。

#### 5.2.4 任务学习视图

点击任务后，场景详情视图隐藏，切换为任务学习视图。采用单页双面板（上下排列）布局。

##### A. 顶部任务信息栏

- **返回场景按钮**：左侧，带返回箭头图标，点击返回场景详情视图。
- **任务名称**：大字号加粗展示。
- **任务元信息**：预估学时 · 难度星级 · 任务类型（考核/训练），右对齐。

三个元素纵向排列：返回按钮独占第一行，任务名称与元信息在同一行紧随其后。

##### B. 学习面板（左侧主区域）

以纵向卡片流展示任务的学习内容：

1. **任务说明卡片**：
   - 卡片头部：「任务说明」标题 + 全屏查看按钮。
   - 卡片内容：富文本渲染的任务详细描述/背景/目标/交付标准，带垂直滚动条（最大高度 420px）。
   - 全屏弹窗：点击全屏按钮后以遮罩弹窗形式展示完整任务说明，支持关闭。

2. **知识点标签**：以可点击 Tag 形式展示本任务关联的知识点，点击后弹出知识点详情卡片。

3. **能力点标签**：以可点击 Tag 形式展示本任务关联的能力点，点击后弹出能力点详情卡片。

4. **交付要求**：
   - 标题：「交付要求」。
   - 列表：每个交付物为一个编号条目，含交付物名称 + 描述。
   - 现场评审类型交付物：带特殊图标标识。

5. **配套资源**：
   - 按资源类型分组展示，同「资源中心」的卡片样式。
   - 支持预览视频/文档、打开外部链接。

##### C. 测评面板（与学习面板上下排列）

当任务配置了测评形式时，在学习面板下方以分割线隔开展示测评内容：

- **测评标题**：「任务测评」+ 图标。
- **测评头部信息**：共 N 道题目，总分 X 分 + 测评类型标签（试卷/题库）。
- **已作答状态**：已提交后展示「已作答」绿色标识。

**各测评形式渲染规则：**

| 测评形式 | 交互说明 |
|---------|---------|
| 试卷测评 | 展示固定试卷题目（单选/多选/判断），每题含选项、分值。已提交后展示正误对比与正确答案。 |
| 题库测评 | 从题库抽取题目，交互形式同试卷测评。 |
| 现场问答 | 展示预约提示：需按预约时间参加，教师将现场抽题提问。 |
| 教师评审 | 展示提交材料要求与评审流程说明，提示在交付步骤中上传成果。 |
| 综合测评 | 当任务配置多种测评形式时，以网格卡片分别展示各测评形式的入口与说明。 |

**题目卡片交互（试卷/题库）：**
- 每题为独立卡片，含题号、题干、分值。
- 选项为可点击行，选中后显示勾选标记。
- 已提交后：正确选项绿色边框+背景，错误选项红色边框+背景，底部显示反馈文案（回答正确/回答错误 + 正确答案）。
- 未提交时底部居中展示「确认答案」主按钮。

##### D. 底部提交栏

位于测评面板下方，根据提交状态展示不同内容：

**未提交状态：**
- 提交内容检查区：列出各测评形式的完成状态摘要（客观题作答状态、成果材料状态、现场问答状态）。
- 学习进度：已完成步骤数 / 总步骤数。
- 警告提示：黄色背景卡片，提示提交后进入「待评分」状态，无法再次修改。
- 「确认提交给教师评分」主按钮。

**已提交状态：**
- 成功图标 + 「已提交给教师评分」标题。
- 说明文案：提示已转交教师评分，可在学习记录中查看状态。
- 提交内容摘要卡片：展示各项测评的提交状态与提交时间。

##### E. 全局资源侧边栏

右侧固定滑出面板，默认收起，通过右下角标签栏触发：

- **面板头部**：标题（配套资源 / 知识点 / 任务）+ 关闭按钮。
- **面板内容**：根据当前激活的 Tab 动态渲染对应的资源/知识点/任务列表。
- **标签栏**（固定在页面右下角）：三个图标按钮——资源、知识、任务，点击切换侧边栏内容并展开面板。

##### F. 模拟页面容器

当任务配置了 simulation 模拟任务时，隐藏原生学习布局，以全屏 iframe 嵌入模拟页面：
- iframe 占满主内容区，监听内部返回消息以退出模拟。

#### 5.2.5 弹窗交互详情

##### A. 任务说明全屏弹窗
- **触发**：点击任务说明卡片的「全屏查看」按钮。
- **内容**：以遮罩层 + 居中弹窗形式展示完整任务说明，弹窗宽度最大 900px，高度 90vh，支持垂直滚动。
- **操作**：点击遮罩层或关闭按钮退出全屏。

##### B. 知识点详情卡片
- **触发**：点击任务学习面板中的知识点标签，或知识图谱中的知识点节点。
- **内容**：知识点名称、描述、编码、关联颗粒课列表。
- **操作**：点击卡片外部关闭。

##### C. 能力点详情卡片
- **触发**：点击任务学习面板中的能力点标签，或知识图谱中的能力点节点。
- **内容**：能力点名称、描述、所属领域、关联岗位、掌握程度要求。
- **操作**：点击卡片外部关闭。

##### D. 资源预览弹窗
- **触发**：点击资源卡片上的「预览」按钮。
- **内容**：视频播放器或文档预览器（本期以占位提示实现）。
- **操作**：「关闭」按钮。

##### E. 学测模式选择弹窗
- **触发**：点击任务列表的「开始学习」按钮（当任务配置了 simulation 时）。
- **内容**：学测一体 / 仅学习 / 仅测评 三种模式选择卡片。
- **操作**：选择模式后进入对应视图，或取消关闭弹窗。
